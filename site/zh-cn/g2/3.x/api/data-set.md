# DataSet

全局命名空间 `DataSet`，同时也是数据集类的构造函数。

## 常量 Constants



## new DataSet()

创建一个数据集实例，返回 DataSet 对象。具体参数见示例代码。

> 示例

```js
const ds = new DataSet({
    state: { // 指定初始化状态量
        foo: 'bar'
    }
});
```
